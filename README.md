# HomePage

## Rules
- use HTML5, CSS3, Javascript(ES6+)
- git commit message

## Project Structure
```
cogito21g.github.io/
|-- index.html
|-- styles.css
|-- main.js
|-- {page_name}/
    |-- {page_name}_index.html
    |-- {page_name}_styles.css
    |-- {page_name}_main.js

```
- README.md: 프로젝트에 관련된 설명
- index.html: 페이지의 구조를 정의. entry point
- styles.css: 페이지의 표현을 정의
- main.js: 페이지의 동적 표현을 정의
- {page_name}/: 페이지별 폴더 생성
